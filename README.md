# Tiny House Nation: Builder Tycoon

Welcome to the Tiny House Nation! In this addictive idle/clicker game, your goal is to build a massive movement of tiny homes across the country.

## How to Play
1.  **Click to Build**: Tap the big green button to construct your tiny house.
2.  **Watch it Grow**: See your house go from framing to finishing touches on the canvas.
3.  **Earn Money**: Each completed house grants you funds.
4.  **Upgrade**: Buy better tools, hire volunteers, and use prefab materials to build faster.
5.  **Expand**: Unlock new states (Oregon, Texas, California, etc.) to access new, more expensive house models.

## Features
- **Educational**: Learn about tiny house technologies (Solar, Compost, Greywater) through upgrades.
- **Progression**: Unlock 5 different house types and expand to 5 different regions.
- **Addictive Loop**: Classic incremental game mechanics with satisfying visual feedback.

## Deployment
This is a static web game (HTML/CSS/JS). Simply host the files on any web server or GitHub Pages.
